# Certean Monitor - Subscription Tiers

## Free Tier
- **Price**: $0/month
- **Features**:
  - 3 products per month
  - Basic compliance checks (Steps 0-2)
  - Email support
  - 7-day data retention

## Professional Tier
- **Price**: $49/month
- **Features**:
  - 50 products per month
  - Full pipeline (Steps 0-5)
  - Priority email support
  - 90-day data retention
  - Export reports (PDF)
  - API access

## Enterprise Tier
- **Price**: $199/month
- **Features**:
  - Unlimited products
  - Full pipeline (Steps 0-5)
  - Dedicated support
  - Unlimited data retention
  - Custom integrations
  - Advanced analytics
  - Multi-user access
  - White-label options
